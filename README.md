# La suerte

Semplice web app in plain HTML, CSS e JavaScript per compiere interrogazioni a sangue.

A cura di [@Nickolausen](https://github.com/Nickolausen), seguendo la scia del prof. [Gabriele Monti](https://github.com/gamondue). 
Nato da un discorso in abilitazione con [@luca-casadei](https://github.com/luca-casadei).

## Dettagli
La repository dispone di 3 branch:
- `main`: (errore di gioventù di @Nickolausen) — codice da manuale di tutte le pratiche da NON seguire per sviluppare un'applicazione: credo che neanche uno dei princìpi SOLID venga rispettato, nè l'idea di 'Separation of concern'
- `dev/mvc`: (capolavoro di @Nickolausen) stessa applicazione, ma in architettura Model-View-Controller
- `dev/clean` (in via di sviluppo, a cura di @luca-casadei) stessa applicazione, ma nella favolosa CLEAN architecture di Uncle Bob (Robert C. Martin)

Potenzialmente didatticamente utile ;-)
